This is a small software project created while learning some basic python and network orchestration concepts.
It is called 'firmware upgrader' even though it does not really do any firmware upgrades (yet).

It uses:
  • Flask with Bootstrap CSS and flask-wtf for forms
  • Nornir (and Napalm) for fetching firmware versions from multiple switches
  • Cisco's OpenVuln API for querying firmware vulnerabilities
  • Netbox IPAM (running as docker containers)
  • Logging
  • Regex
  • Python Threads and Queues (for working off long running jobs in the background)
