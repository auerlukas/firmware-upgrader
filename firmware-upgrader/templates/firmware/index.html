Firmware

Uses nornir's parralel computing feature to get firmware of all devices belonging to site 'cisco' (see devices.yaml)

fetch firmware versions

This will add the variable 'firmware' to the nornir object/inventory which can then be accessed via nr.inventory.hosts[hostname].firmware
Note: This will start a long running job, put this job into the job manager queue (running in its own thread).
Once the job complets, the user can click on "Inventory" in the menu and will see an updated inventory including firmware versions.
